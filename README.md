# Data-Profession-Survey-Dashboard
Overview : 
This Power BI dashboard analyzes survey responses from data professionals across various roles, including Data Analysts, Data Scientists, Data Engineers, Database Developers and other data-related careers. The project explores key aspects of the data industry such as salary trends, work-life balance, job satisfaction, preferred programming languages and regional differences.

Goal :
The goal of this dashboard is to transform survey data into actionable insights that help understand the experiences, preferences and compensation patterns of professionals working in the data field.

Objectives :
This project aims to:
- Explore salary variations across different data professions.
- Compare work-life balance ratings among data professionals.
- Analyze job satisfaction and career happiness.
- Identify the most popular programming languages used in the industry.
- Examine geographical trends and regional differences within the data profession landscape.
- Provide an interactive and visually engaging dashboard for data exploration.

Tools & Technologies : 
- Power BI – Data visualization and dashboard development
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – Calculated measures and KPIs
- Excel/CSV – Source data storage
- Dashboard Features

The dashboard includes:
- Salary analysis by job title
- Work-life balance ratings
- Job satisfaction insights
- Geographic distribution of respondents
- Favorite programming language by profession
- Interactive filters and metric cards for deeper analysis

Key Insights :
Some notable findings from the survey include:
- Salary levels vary significantly across data-related professions.
- Work-life balance ratings differ between roles and regions.
- Certain programming languages are more popular within specific data careers.
- Geographic location plays a role in compensation and professional experiences.
